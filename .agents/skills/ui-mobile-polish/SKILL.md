# 역할
모바일 게임 느낌의 UI 사용성을 개선한다.

# 언제 쓰나
- 기능은 되지만 화면이 어색할 때
- PlayPage / ResultPage / HomePage 사용성을 다듬을 때
- 모바일 터치 영역을 조정할 때

# 핵심 작업 지침
1. 모바일 화면 기준으로 먼저 본다.
2. 버튼은 큰 터치 영역을 유지한다.
3. 상단 HUD는 아래 정보를 우선 보이게 한다.
   - 하트
   - 점수
   - 콤보
   - 타이머
4. 카드형 문제 UI를 기본으로 유지한다.
5. 교육 사이트 느낌보다 게임 UI 느낌을 우선한다.
6. 텍스트가 너무 작거나 버튼이 빽빽하면 수정한다.
7. PC 대응은 모바일 구조를 해치지 않는 선에서 확장한다.

# 점검 항목
- 터치 영역 충분 여부
- 텍스트 가독성
- 상단 상태 정보 노출
- 화면 전환 흐름
- 세로 화면 기준 안정성
