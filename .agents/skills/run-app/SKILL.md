# 역할
로컬 실행과 기본 UI 동작을 빠르게 확인한다.

# 언제 쓰나
- 초기 세팅 직후
- 기능 구현 직후
- 배포 전 기본 동작 점검 시

# 핵심 작업 지침
1. 설치가 안 되어 있으면 먼저 `npm install` 상태를 확인한다.
2. 개발 서버 실행 명령을 확인한다.
3. 아래 핵심 화면을 우선 확인한다.
   - 로그인
   - 언어 선택
   - 홈
   - 플레이
   - 결과
4. 콘솔 에러가 있으면 먼저 기록한다.
5. 문제 발생 시 전체 수정 전에 가장 작은 재현 경로를 찾는다.
6. 결과는 간단히 정리한다.
   - 실행 성공/실패
   - 확인한 화면
   - 발견한 오류
   - 다음 조치

# 기본 명령 예시
- npm install
- npm run dev
