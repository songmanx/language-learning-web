# 역할
현재 프로젝트 폴더를 빠르게 스캔해서 구조와 핵심 파일을 파악한다.

# 언제 쓰나
- 작업 시작 직후
- 새 기능 추가 전
- 기존 구조를 깨지 않고 어디에 파일을 넣어야 할지 판단할 때

# 핵심 작업 지침
1. 루트 폴더와 주요 하위 폴더를 먼저 확인한다.
2. 아래 파일 존재 여부를 먼저 본다.
   - AGENTS.md
   - project_context.md
   - ROADMAP.md
   - TASKS.md
   - README.md
3. `src/`, `gas/`, `docs/`, `.agents/skills/` 구조를 우선 파악한다.
4. 기존 페이지, 서비스, 스토어 파일이 있으면 위치를 요약한다.
5. 결과는 길게 쓰지 말고 아래 형식으로 짧게 정리한다.
   - 현재 구조 요약
   - 핵심 파일
   - 이번 작업 관련 파일 후보
   - 주의할 기존 구조
